# 역추적 인덱스 — {서비스명}

> 설계도/문제점 리포트에서 코드 위치를 빠르게 찾기 위한 역추적 인덱스입니다.
> 각 Step 산출물 작성 시 이 인덱스에 주요 항목을 등록하세요.
> 소스 로케이터 포맷: [source-locator-standard.md](../../../../project/source-locator-standard.md) 참조

## 사용 방법

1. PM이 설계도/Gap 리포트에서 특정 항목의 코드를 확인하고 싶을 때 이 인덱스를 검색
2. 항목명, 화면 ID, API 경로, 비즈니스 규칙 ID 등으로 검색하여 소스 위치를 즉시 파악
3. 소스 위치의 Repo명/경로를 IDE에서 열어 코드 확인
