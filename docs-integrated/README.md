# 통합 설계도 (Integrated Documentation)

모든 서비스의 개별 분석이 완료된 후, 이 폴더에 서비스별 섹션으로 구분된 통합 문서를 생성합니다.

Confluence 등 문서 포털에 복붙할 때 이 통합본을 사용합니다.

## 폴더 구조

```
docs-integrated/
├── extraction/      ← Phase 1 통합본 (서비스별 섹션 구분)
├── views/           ← Phase 2 통합본 (서비스 횡단 View 포함)
├── gap-analysis/    ← Phase 3 통합본
└── glossary/        ← 용어 사전 통합본
```
