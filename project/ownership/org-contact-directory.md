# 조직 연락처 디렉토리 (Org Contact Directory)

> 조직(팀/파트) → 현재 담당자(PM), 외주사 연락처 매핑.
> 조직재편 시 이 파일만 갱신하면 된다. 코드 소유권 매핑은 [code-ownership-map.md](code-ownership-map.md) 참조.

---

## 갱신 규칙

1. 조직재편 발생 시 이 파일만 갱신 (code-ownership-map.md는 팀명이 변경된 경우에만 갱신)
2. 각 행에 `유효기간` 컬럼을 기재하여 정보의 신선도를 관리
3. 담당자 변경 시 이전 담당자를 `이전 담당` 컬럼으로 이동 (이력 보존)
4. 최소 반기 1회 전체 갱신 검토
